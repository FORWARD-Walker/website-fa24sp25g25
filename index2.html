What is FORWARD?

Feedback Oriented Routing and Walker Assistance with Responsive Direction

"FORWARD autonomously guides the user through their environment, providing access to the outside world like never before."

1

Obstacle Detection

Notify the user of obstacles ahead. Sensor fusion allows for the most accurate ranging data.

2

Object Identification

Computer Vision and image processing allow FORWARD to classify the danger level of detected hazards.

3

Obstacle Avoidance

FORWARD provides haptic and audio feedback to guide the user, while also providing steering and stability assistance.
